Introduction

Welcome to the Mobtakir API. Our API allows you to integrate AI learning capabilities directly into your applications.

Base URL
https://api.aimaster.com/v1

Authentication

All API requests require authentication using an API key in the header: 

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /users/{user_id}

Retrieve user information

Parameters
Parameter Type Description
user_id Required string The unique identifier of the user 
curl -X GET "https://api.aimaster.com/v1/users/123" \
-H "Authorization: Bearer YOUR_API_KEY"
Response Example
{
    "id": "123",
    "name": "John Doe",
    "email": "john@example.com",
    "created_at": "2024-01-01T00:00:00Z"
}
POST /courses

Create a new course

curl -X POST "https://api.aimaster.com/v1/courses" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "title": "Advanced AI Course",
    "description": "Learn advanced AI concepts",
    "duration": 120
}'

Error Handling

The API uses conventional HTTP response codes to indicate success or failure of requests.

Code Description
200 Success
400 Bad Request
401 Unauthorized
404 Not Found
429 Too Many Requests
500 Server Error

Rate Limiting

API requests are limited to 100 requests per minute per API key. Rate limit information is included in the response headers: 

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1640995200
